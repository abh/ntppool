Das pool.ntp.org-Projekt ist ein großer, virtueller Cluster aus Zeitservern, die einen verläßlichen und einfach zu verwendenden NTP-Dienst für millionen Clients anbieten.

Der Pool wird von millionen oder zehn millionen Systemen auf der ganzen Welt benutzt. Es ist der standard Zeitserver der meisten großen Linux-Distributionen und vieler netzwerkbasierter Anwendungen (siehe Informationen für Händler).

Aufgrund der großen Anzahl Nutzer/innen brauchen wir mehr Server. Wenn du einen Server mit einer statischen IP-Adresse hast und eine permanente Internetverbidnung, bitte überlege dir, dein System zum Pool hinzuzfügen.

Das Projekt wird unterhalten und entwickelt von Ask Bjørn Hansen und einer großen Gruppe Unterstützer/innen in den mailing-Listen. Der Code ist frei verfügbar.

Hosting und Bandbreite für die "hub"-Server werden derzeit durch Develooper, Phyber Communications und YellowBot zur Verfügung gestellt.
