Die Bewertungsgrafik

Mehrmals pro Stunde prüft das Pool-System deinen Server und vergleicht ihn mit der lokalen Uhrzeit. Punkte werden abgezogen, wenn der Server nicht erreicht werden kann oder der Zeitunterschied mehr als 100 ms ist (relativ grob gemessen durch das System). Je größer der Unterschied, desto mehr Punkte werden abgezogen.

Wenn die Wertung sinkt, zeigt die Hintergrundfarbe im oberen Teil, wie schwerwiegend der "Ausfahl" ist. Die Farbskala geht von Blau (sehr gering) zu Gelb und Orange zu Rot (der Server ist mehrere Sekunden nicht erreichbar oder aus). Weil die Grafik gemittelt wird, kann man anhand der Farbe nicht immer sagen, was passiert ist. Es ist nur als ein Tool um Trends anzuzeigen gedacht. Für exakte Details, was das Monitor-System fand, kannst du den CSV-Link anklicken.

Die Abweichungsgrafik

Das Monitor-System funktioniert etwa wie ein SNTP (RFC 2030)-Client. Es ist also empfindlicher auf zufällige Netzwerklatenz zwischen Server und dem System als ein normaler ntpd-Client sein würde. In anderen Worten: Sei nicht verunsichert durch eine einstweilige, große Abweichung und bitte benutze die Abweichung nich als absolute Perfomancemessung.
