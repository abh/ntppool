Das Punktediagram

Mehrmals pro Stunde wird die Zeit Ihres Zeitservers mit der tatsächlichen Zeit verglichen. Sollte Ihr Server nicht erreichbar sein, oder geht Ihre Uhr mehr als 100 ms falsch, werden Ihnen Punkte abgezogen. Für größere Fehler werden mehr Punkte abgezogen.

Wenn Ihr Punktestand fällt, zeigt Ihnen die Hintergrundfarbe in der oberen Hälfte des Diagrams die Schwere des Punkteabzugs an. Die Farbskala geht von blaub für sehr kleine Fehler über gelb und orange bis rot, falls der Zeitfehler mehrere Sekunden beträgt oder Ihr Server nicht erreichbar ist. Durch die eingesetzte Mittelung kann es teilweise sehr schwer werden auf die Ursache des Punkteabzugs zu schließen. Das Diagramm dient lediglich als Werkzeug zur Visualisierung eines Trends. Die exakten Ergebnisse des Überwachungssystems können als CSV-Datei heruntergeladen werden.

Das Versatzdiagram

Das Überwachungssystem funktioniert etwa wie ein SNTP (RFC 2030) Client, so dass es viel empfindlicher auf Netzwerklatenzen zwischen Server und Überwachungssystem reagiert, als ein normaler ntpd-Dienst. In anderen Worten: Die Diagramme können nicht zur absoluten Leistungsmessung des Zeitservers eingesetzt werden.
