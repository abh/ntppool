Das Punktediagram

Mehrere Male in der Stunde wird die Zeit ihres Zeitservers mit der tatsächlichen Zeit verglichen. Sollte Ihre Server nicht erreichbar sein, oder geht Ihre Uhr mehr als 100ms falsch, werden Ihnen Punkte abgezogen. Für größere Fehler werden mehr Punkte abgezogen.

Wenn ihr Punktestand fällt, zeigt Ihnen die Hintergrundfarbe in der oberen Hälfte des Diagrams die Schwere des Punkteabzugs an. Die Farbskala geht von blaub für sehr kleine Fehler über gelb und orange bis rot, falls der Zeitfehler mehre Sekunden beträgt oder Ihr Server nicht erreichbar ist. Durch die eingesetzte Mittelung kann es teilweise sehr schwer werden auf die Ursache des Punkteabzugs zu schließen. Das Diagram dient lediglich als Werkzeug zur Visualisierung eines Trends. Die exakten Ergebnisse des Überwachungssystems können als CSV-Datei heruntergeladen werden.

Das Versatzdiagram

Das Überwachungssystem funktioniert grob wie ein SNTP (RFC 2030) Klient, so dass es sehr empfindlich auf Netzwerklatenzen zwischen Server und Überwachungssystem reagiert als ein normaler ntpd-Dienst. In anderen Worten: Die Diagramme können nicht zur absolute Leistungsmessung des Zeitservers eingesetzt werden.
