Et par gange i timen vil systemet checke din servers tid og sammenligne den med lokal tid. Der trækkes point hvis din server ikke kan tilgås eller hvis forskellen er mere end 100ms. Jo større forskel der er jo flere point trækkes der.

Grafen er kun ment som et værktøj til at visualisere tendenser. For mere nøjagtige detaljer, klik da på CSV linket.
