El gráfico de puntuación

Un par de veces por hora, el sistema de monitorización comprueba la hora de tu servidor y la compara con la hora local. La puntuación baja si no se puede contactar con el servidor o si la diferencia de tiempo es superior a 100ms (como se miden los servidores de monitorización). Esta puntuación baja más cuanto mayor sea la diferencia.

Está pensada como una herramienta para visualizar tendencias. Para obtener datos más exactos acerca de lo que ha encontrado el sistema de monitorización puedes hacer clic en el enlace CSV.

El gráfico de diferencias

El sistema de monitorización trabaja básicamente como un cliente SNTP ( RFC 2030), por lo que es susceptible de obtener má latencias de red aleatorias entre el servidor y el sistema de monitorización de lo que obtendría un cliente ntpd. En otras palabras: no te alarmes si obtienes una gran diferencia en algunas ocasiones y por favor, no utilices esta diferencia como una métrica absoluta de rendimiento.
