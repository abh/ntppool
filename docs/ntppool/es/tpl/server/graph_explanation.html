El gráfico de puntuación

Un par de veces por hora, el sistema del pool comprueba la hora de tu servidor y la compara con la hora local. Se deducen puntos si no se puede contactar con el servidor o si la diferencia de tiempo es superior a 100ms (como se miden desde los sistemas de monitorización). Se deducen más puntos cuanto mayor sea la diferencia.

El gráfico está pensado únicamente como una herramienta para visualizar tendencias. Para obtener datos más exactos acerca de lo que ha encontrado el sistema de monitorización puedes hacer clic en el enlace CSV.

El gráfico de diferencias

El sistema de monitorización trabaja básicamente como un cliente SNTP ( RFC 2030), por lo que es más susceptible a latencias de red aleatorias entre el servidor y el sistema de monitorización de lo que sería un servidor ntpd regular.

El sistema de monitorización puede ser inexacto hasta 10ms o más.
