โครงการ pool.ntp.org เป็นกลุ่มเซิร์ฟเวอร์เวลาเสมือนขนาดใหญ่ที่ให้บริการ NTP ที่เชื่อถือได้และใช้งานง่ายสำหรับไคลเอนต์หลายล้านราย

พูลนี้ถูกใช้งานโดยระบบหลายร้อยล้านระบบทั่วโลก โดยเป็นเซิร์ฟเวอร์เวลาเริ่มต้นสำหรับลีนุกซ์ดิสทริบิวชันหลักส่วนใหญ่ และอุปกรณ์เครือข่ายจำนวนมาก (ดูข้อมูลสำหรับผู้จำหน่าย)

เนื่องจากมีผู้ใช้จำนวนมาก เราจึงต้องการเซิร์ฟเวอร์เพิ่มเติม หากคุณมีเซิร์ฟเวอร์ที่มี IP แอดเดรสคงที่และเชื่อมต่ออินเทอร์เน็ตตลอดเวลา กรุณาพิจารณาเพิ่มเซิร์ฟเวอร์ของคุณเข้าร่วมโครงการ

โครงการนี้ดูแลและพัฒนาโดย Ask Bjørn Hansen และกลุ่มผู้ร่วมพัฒนาที่ยอดเยี่ยมในฟอรัมชุมชนของโครงการ โดยซอร์สโค้ดของระบบเปิดให้เข้าถึงได้

การโฮสต์และแบนด์วิดท์สำหรับเซิร์ฟเวอร์ฮับ ได้รับการสนับสนุนจาก Equinix และ Netactuate
