अनुवादकों की आवश्यकता है!

हम एनटीपी पूल वेबसाइट की मूल सामग्री लगभग सभी भाषाओं में उपलब्ध कराना चाहते हैं; लेकिन इसके लिए हमें आपकी मदद चाहिए!

आपको किन चीज़ों की आवश्यकता होगी

  • आपको अंग्रेजी का सक्षम पाठक और किसी अन्य भाषा में पारंगत होना चाहिए।
  • प्रारंभिक अनुवाद करने के लिए 3-4 घंटे का समय।
  • नए परिवर्तनों के अनुवाद को अपडेट करने के लिए हर 3 महीने में लगभग एक या दो बार 10-20 मिनट का समय।
  • HTML का कुछ बुनियादी ज्ञान।
  • Git या Subversion की बुनियादी जानकारी।
  • हज़ारों लोगों के लाभ के लिए मुफ़्त में काम करने की इच्छा!

कैसे शुरू करें

आरंभ करने का सबसे आसान तरीका है Git के साथ कोड को क्लोन करना। Git का उपयोग करने के लिए पहले Github पर अकाउंट बनाएं और फिर ntppool रिपॉजिटरी को क्लोन (या फोर्क) करें।

कृपया ntppool फोरम के लिए साइन अप करें और (बहुत कम ट्रैफिक वाली) अनुवादकों और i18n श्रेणी की सदस्यता लें।

अनुवाद करें

जब आपके पास स्रोत कोड होगा, तो आपको निम्न दो स्थानों पर अनुवाद करने के लिए स्ट्रिंग मिलेंगी:

  • i18n/ फ़ोल्डर में .po फ़ाइलें। आप इन फाइलों को किसी भी टेक्स्ट एडिटर (उदाहरण के लिए POEDIT) से एडिट कर सकते हैं। यदि आपकी भाषा के लिए कोई फ़ाइल नहीं है तो आप "en.po" फ़ाइल की कॉपी बनाकर शुरू कर सकते हैं।
  • docs/ntppool/ फ़ोल्डर में भाषा विशिष्ट HTML फ़ाइलें। केवल "भाषा फ़ोल्डर" की फ़ाइलों को ही अनुवाद की आवश्यकता है। बाकी या तो केवल अंग्रेजी में उपलब्ध कराए गई हैं या .po फ़ाइलों के माध्यम से अनुवादित हैं। सबसे महत्वपूर्ण फ़ाइलें हैं: "homepage.html", "join.html" और "use.html"।
  • हो सकता है "beta" में अधूरा अनुवाद उपप्लब्ध हो (टेस्ट साइट पर 'beta' भाषाएं देखें)।

अनुवाद सबमिट करें

जब अनुवाद पूरा हो जाये तो आप या तो git से कमिट कर सकते हैं या github पर pull रिक्वेस्ट कर सकते हैं। अगर आपके कोई सवाल हैं तो आप उनको अनुवादक फोरम में पोस्ट कर सकते हैं।

बीटा टेस्ट

शुरआत में नयी भाषा बीटा साइट पर दर्शायी जाएगी। बीटा साइट पर जाकर समीक्षा करें और अगर संभव हो तो अपनी भाषा में पारंगत किसी अन्य व्यक्ति से भी समीक्षा कराएं। उसके बाद github या अनुवादक फोरम पर जाकर फीडबैक दें कि अनुवाद मुख्य साइट पर जाने के लिए तैयार है।
