O gráfico de pontuação

Algumas vezes em uma hora o sistema de pool verifica a hora a partir do seu servidor e compara-a com o horário local. Pontos são deduzidos se o servidor não puder ser alcançado ou se a diferença de hora for maior do que 100ms (como é medido relativamente a grosso modo dos sistemas de monitoramento). Mais pontos são deduzidos na medida em que aumenta a diferença.

Foi feito para ser uma ferramenta para visualizar tendências. Para detalhes exatos do que o sistema de monitoramento encontrou, basta clicar no link de CSV.

O gráfico de diferença

O sistema de monitoramente funciona basicamente como um cliente SNTP (RFC 2030), de forma que ele é mais suscetível a latências aleatórias da rede entre servidor e sistema de monitoramento do que um cliente ntpd comum estaria. Em outras palavras: não se preocupe com grandes diferenças ocasionais e por favor não use a diferença como uma métrica absoluta de performance.
