Πώς μπορώ να χρησιμοποιήσω το pool.ntp.org;

Αν απλά θέλετε να συγχρονίσετε το ρολόι του υπολογιστή σας με το δίκτυο, το αρχείο διαμόρφωσης (για το πρόγραμμα ntpd από τη διανομή ntp.org, σε οποιοδήποτε υποστηριζόμενο λειτουργικό σύστημα - Linux, *BSD, Windows και ακόμη και μερικά πιο εξωτικά συστήματα) είναι πραγματικά απλό:

[% INCLUDE "ntppool/use/sample-config.html" %]

Τα ονόματα 0, 1, 2 και 3.pool.ntp.org δείχνουν σε ένα τυχαίο σύνολο διακομιστών που αλλάζουν κάθε ώρα. Βεβαιωθείτε ότι το ρολόι του υπολογιστή σας έχει ρυθμιστεί σε κάτι λογικό (μέσα σε λίγα λεπτά από την 'πραγματική' ώρα) - θα μπορούσατε να χρησιμοποιήσετε το ntpdate pool.ntp.org, ή θα μπορούσατε απλώς να χρησιμοποιήσετε την εντολή date και να το ρυθμίσετε σύμφωνα με το ρολόι χειρός σας. Ξεκινήστε το ntpd, και μετά από κάποιο χρόνο (αυτό μπορεί να διαρκέσει έως και μισή ώρα!), η εντολή ntpq -pn θα πρέπει να δώσει κάτι παρόμοιο:

[% INCLUDE "ntppool/use/sample-ntpq.html" %]

Οι διευθύνσεις IP θα είναι διαφορετικές, επειδή σας έχουν ανατεθεί τυχαίοι διακομιστές χρόνου. Το ουσιώδες είναι ότι μία από τις γραμμές ξεκινά με αστερίσκο (*), αυτό σημαίνει ότι ο υπολογιστής σας παίρνει τον χρόνο από το Διαδίκτυο - δεν θα χρειαστεί ποτέ να ανησυχήσετε γι' αυτό ξανά!

Η αναζήτηση του pool.ntp.org (ή 0.pool.ntp.org, 1.pool.ntp.org, κλπ.) θα επιστρέψει συνήθως διευθύνσεις IP για διακομιστές στη χώρα σας ή κοντά σε αυτήν. Για τους περισσότερους χρήστες αυτό θα δώσει τα καλύτερα αποτελέσματα.

Μπορείτε επίσης να χρησιμοποιήσετε τις ηπειρωτικές ζώνες (Για παράδειγμα europe, north-america, oceania ή asia.pool.ntp.org), και μια ζώνη χώρας (όπως ch.pool.ntp.org στην Ελβετία) - για όλες αυτές τις ζώνες, μπορείτε ξανά να χρησιμοποιήσετε τα προθέματα 0, 1, 2, ή 3, όπως 0.ch.pool.ntp.org. Σημειώστε, ωστόσο, ότι η ζώνη της χώρας μπορεί να μην υπάρχει για τη χώρα σας, ή μπορεί να περιέχει μόνο έναν ή δύο διακομιστές χρόνου.

Παρακαλούμε επίσης σημειώστε ότι το σύστημα προς το παρόν παρέχει διευθύνσεις IPv6 για μια ζώνη επιπρόσθετα των διευθύνσεων IPv4 μόνο αν το όνομα της ζώνης προηγείται από τον αριθμό 2, π.χ. 2.pool.ntp.org (υπό την προϋπόθεση ότι υπάρχουν διακομιστές NTP IPv6 στην αντίστοιχη ζώνη). Ονόματα ζωνών που δεν προηγούνται από αριθμό, ή προηγούνται από 0, 1 ή 3, παρέχουν προς το παρόν μόνο διευθύνσεις IPv4.

Αν χρησιμοποιείτε μια πρόσφατη έκδοση των Windows, μπορείτε να χρησιμοποιήσετε τον πελάτη ntp που είναι ενσωματωμένος στο σύστημα.

Κάντε δεξί κλικ στην ημερομηνία/ώρα στην περιοχή συστήματος και επιλέξτε Προσαρμογή Ημερομηνίας/ώρας ή ανοίξτε τις ρυθμίσεις με Win+I και κάντε κλικ Ώρα & Γλώσσα -> Ημερομηνία & Ώρα. κυλήστε προς τα κάτω και κάντε κλικ Ημερομηνία, ώρα, περιφερειακή μορφοποίηση κυλήστε προς τα κάτω και κάντε κλικ Πρόσθετες ρυθμίσεις ημερομηνίας, ώρας & περιφερειακές κάντε κλικ Ημερομηνία και Ώρα και Ώρα Διαδικτύου κάντε κλικ στο κουμπί Αλλαγή ρυθμίσεων. Εισάγετε pool.ntp.org και κάντε κλικ Ενημέρωση τώρα.

Δοκιμάστε κάνοντας δεξί κλικ στην ημερομηνία/ώρα στην περιοχή συστήματος και επιλέξτε Προσαρμογή Ημερομηνίας/ώρας ή ανοίξτε τις ρυθμίσεις με Win+I και κάντε κλικ Ώρα & Γλώσσα -> Ημερομηνία & Ώρα. Κάντε κλικ στο κουμπί "Συγχρονισμός τώρα".

Αν το σύστημα Windows σας είναι μέρος ενός domain, μπορεί να μην μπορείτε να ενημερώσετε ανεξάρτητα τον χρόνο του υπολογιστή σας. Για περισσότερες πληροφορίες σχετικά με τη ρύθμιση της ώρας στα Windows, δείτε How Windows Time Service Works.

Πρόσθετες Σημειώσεις

Σκεφτείτε αν η Δεξαμενή NTP είναι κατάλληλη για τη χρήση σας. Αν η επιχείρηση, ο οργανισμός ή η ανθρώπινη ζωή εξαρτάται από το να έχετε σωστό χρόνο ή μπορεί να βλαφτεί αν είναι λάθος, δεν θα πρέπει να "το παίρνετε απλώς από το Διαδίκτυο". Η Δεξαμενή NTP είναι γενικά πολύ υψηλής ποιότητας, αλλά είναι μια υπηρεσία που λειτουργεί από εθελοντές στον ελεύθερο χρόνο τους. Παρακαλούμε μιλήστε με τους προμηθευτές εξοπλισμού και υπηρεσιών σας για να αποκτήσετε τοπική και αξιόπιστη ρύθμιση υπηρεσίας. Δείτε επίσης τους όρους υπηρεσίας μας. Συνιστούμε διακομιστές χρόνου από τη Meinberg, αλλά μπορείτε επίσης να βρείτε διακομιστές χρόνου από την End Run, Spectracom και πολλούς άλλους.

Αν έχετε στατική διεύθυνση IP και λογική σύνδεση Διαδικτύου (το bandwidth δεν είναι τόσο σημαντικό, αλλά θα πρέπει να είναι σταθερό και όχι πολύ φορτωμένο), παρακαλούμε σκεφτείτε να δωρίσετε τον διακομιστή σας στη δεξαμενή διακομιστών. Δεν θα σας κοστίσει περισσότερο από λίγες εκατοντάδες bytes ανά δευτερόλεπτο κίνηση, αλλά βοηθάτε αυτό το έργο να επιβιώσει. Παρακαλούμε διαβάστε τη σελίδα συμμετοχής για περισσότερες πληροφορίες.

Αν ο πάροχος Διαδικτύου σας έχει διακομιστή χρόνου, ή αν γνωρίζετε έναν καλό διακομιστή χρόνου κοντά σας, θα πρέπει να χρησιμοποιήσετε αυτόν και όχι αυτή τη λίστα - θα πάρετε πιθανώς καλύτερο χρόνο και θα χρησιμοποιήσετε λιγότερους δικτυακούς πόρους. Αν γνωρίζετε μόνο έναν διακομιστή χρόνου κοντά σας, μπορείτε φυσικά να χρησιμοποιήσετε αυτόν και δύο από το pool.ntp.org.

Μπορεί σπάνια να συμβεί να σας ανατεθεί ο ίδιος διακομιστής χρόνου δύο φορές - απλώς επανεκκινώντας τον διακομιστή ntp συνήθως λύνει αυτό το πρόβλημα. Αν χρησιμοποιείτε ζώνη χώρας, παρακαλούμε σημειώστε ότι μπορεί να οφείλεται στο ότι υπάρχει μόνο ένας διακομιστής γνωστός στο έργο - καλύτερα χρησιμοποιήστε ηπειρωτική ζώνη σε αυτή την περίπτωση. Μπορείτε να περιηγηθείτε στις ζώνες για να δείτε πόσους διακομιστές έχουμε σε κάθε ζώνη.

Να είστε φιλικοί. Πολλοί διακομιστές παρέχονται από εθελοντές, και σχεδόν όλοι οι διακομιστές χρόνου είναι στην πραγματικότητα διακομιστές αρχείων ή mail ή web που τυχαίνει επίσης να τρέχουν ntp. Οπότε μη χρησιμοποιείτε περισσότερους από τέσσερις διακομιστές χρόνου στη διαμόρφωσή σας, και μη παίζετε τρικ με το burst ή minpoll - το μόνο που θα κερδίσετε είναι επιπλέον φορτίο στους εθελοντικούς διακομιστές χρόνου.

Βεβαιωθείτε ότι η διαμόρφωση ζώνης ώρας του υπολογιστή σας είναι σωστή. Το ίδιο το ntpd δεν κάνει τίποτα για τις ζώνες ώρας, χρησιμοποιεί απλώς UTC εσωτερικά.

Αν συγχρονίζετε ένα δίκτυο με το pool.ntp.org, παρακαλούμε ρυθμίστε έναν από τους υπολογιστές σας ως διακομιστή χρόνου και συγχρονίστε τους άλλους υπολογιστές με αυτόν. (θα έχετε κάποιο διάβασμα να κάνετε - δεν είναι δύσκολο όμως. Και υπάρχει πάντα η ομάδα ειδήσεων comp.protocols.time.ntp.)

Σε αυτό το σημείο, θα ήθελα να ευχαριστήσω αυτούς που δωρίζουν τον χρόνο και τους διακομιστές χρόνου τους σε αυτό το δίκτυο.
