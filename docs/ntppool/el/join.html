[% page.title = 'Συμμετέχετε στη Δεξαμενή NTP!' %]

Πώς μπορώ να συμμετάσχω στο pool.ntp.org;

Πρώτα: Ευχαριστούμε για το ενδιαφέρον σας. Η χρήση της δεξαμενής έχει αυξηθεί τεράστια, και ο μόνος τρόπος να διατηρήσουμε εύκολη τη λειτουργία ενός διακομιστή δεξαμενής για όλους είναι να αυξήσουμε επίσης τον αριθμό των συμμετεχόντων διακομιστών.

Ο υπολογιστής σας πρέπει να έχει στατική διεύθυνση IP και μόνιμη σύνδεση Διαδικτύου. Είναι πολύ σημαντικό η διεύθυνση IP σας να μην αλλάζει ή να το κάνει εξαιρετικά σπάνια (ας πούμε μία φορά το χρόνο ή λιγότερο). Το απαιτούμενο bandwidth είναι σχετικά χαμηλό. Κάθε πελάτης θα στέλνει μόνο λίγα UDP πακέτα κάθε λίγα έως κάθε 20 λεπτά.

Επί του παρόντος οι περισσότεροι διακομιστές λαμβάνουν περίπου 5-15 πακέτα NTP ανά δευτερόλεπτο με αιχμές 60-120 πακέτων ανά δευτερόλεπτο λίγες φορές τη μέρα. Αυτό ισοδυναμεί περίπου με 10-15Kbit/δευτ με αιχμές 50-120Kbit/δευτ. Το έργο αποκτά σταθερά περισσότερους διακομιστές χρόνου, οπότε το φορτίο δεν θα πρέπει να αυξηθεί δραματικά για κάθε διακομιστή. Με απλά λόγια, χρειάζεστε πιθανώς τουλάχιστον 384-512Kbit bandwidth (εισερχόμενο και εξερχόμενο).

Οι διακομιστές που συμμετέχουν δεν θα πρέπει να έχουν pool.ntp.org ως ανάντη διακομιστή(-ες) τους, αλλά θα πρέπει να διαμορφώσουν κάποιους καλούς διακομιστές χειροκίνητα. Αυτοί οι διακομιστές μπορούν να επιλεγούν από τη δεξαμενή. Το σημείο είναι ότι επιλέγονται στατικά αντί να ανατίθενται τυχαία από τη δεξαμενή σε κάθε επανεκκίνηση διακομιστή. Αυτό θα βοηθήσει στην παροχή αποδεκτής ποιότητας υπηρεσίας. Σημειώστε ότι δεν απαιτείται ο διακομιστής σας να είναι stratum 1 ή 2 διακομιστής - καθώς αυτό το έργο αφορά κυρίως τη διανομή φορτίου, δεν υπάρχει λόγος να μη συμμετάσχει ένας διακομιστής stratum 3 ή ακόμη και stratum 4.

Έχουμε μια σελίδα με συστάσεις διαμόρφωσης για διακομιστές που συμμετέχουν στη δεξαμενή.

Τέλος, πρέπει να τονίσω ότι η συμμετοχή στη δεξαμενή είναι μια μακροπρόθεσμη δέσμευση. Χαρούμε να σας αφαιρέσουμε από τη δεξαμενή ξανά αν οι συνθήκες σας αλλάξουν, αλλά λόγω του τρόπου λειτουργίας των πελατών NTP θα χρειαστούν εβδομάδες, μήνες ή ακόμη και ΧΡΟΝΙΑ πριν η κίνηση εξαφανιστεί εντελώς.

Αν όλα αυτά είναι εντάξει, συνδεθείτε στη σελίδα διαχείρισης διακομιστή και ζητήστε τον διακομιστή σας να προστεθεί. Αν έχετε οποιοδήποτε πρόβλημα με το σύστημα, στείλτε μου email στο ask@develooper.com.

Παρακαλούμε επίσης συμμετάσχετε στο φόρουμ κοινότητας

Θα ήταν ωραίο (αλλά δεν είναι απαραίτητο) αν μπορούσατε να ανακατευθύνετε τα αιτήματα web στη θύρα 80 του διακομιστή χρόνου σας στην επίσημη ιστοσελίδα του έργου στο https://www.ntppool.org/. Αν τρέχετε Apache, θα μπορούσατε να το κάνετε αυτό με:

[% PROCESS tpl/join/virtual_host_example.html %]

Πάλι, αυτό είναι μόνο αν τρέχετε έναν webserver ούτως ή άλλως. Η επίσημη ιστοσελίδα του έργου θα δίνεται πάντα με το 'www' στην αρχή - αλλά μερικές φορές οι άνθρωποι πληκτρολογούν pool.ntp.org και στη συνέχεια εκπλήσσονται που παίρνουν μια τυχαία ιστοσελίδα.

Μόλις προστεθεί στη δεξαμενή ο διακομιστής παρακολουθείται συνεχώς για διαθεσιμότητα και ακρίβεια. Μπορείτε να παρακολουθήσετε την απόδοση του διακομιστή σας χρησιμοποιώντας το web frontend ή τη σελίδα διαχείρισης.
