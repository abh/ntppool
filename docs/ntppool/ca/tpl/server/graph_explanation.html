El gràfic de puntuació

Cada hora, el sistema del pool comprova l'hora del vostre servidor i la compara amb l'hora local. Si no s'hi pot connectar o la diferència de l'hora és més de 100ms (mesurada amb la precisió relativament baixa del sistema de vigilància), la puntuació del vostre servidor baixarà. Quant més gran sigui la diferència, més punts es restaran.

La gràfica només es una eina per veure tendències. Per a més detalls sobre els resultats del sistema de vigilància pots fer clic sobre l'enllaç de CSV.

El gràfic de diferència

El sistema de vigilància funciona més o menys com un client SNTP (RFC 2030). Per tant, és més susceptible a canvis de latències de la xarxa que un client ntpd normal.

El sistema de vigilància pot tenir una latència de 10ms o més.
