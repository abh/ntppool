Puntuazio grafikoa

Orduro zenbait aldiz pool sistemak zure zerbitzariko denbora egiaztatzen du eta bere denbora lokalarekiko konparatzen du. Puntuak kentzen dira zerbitzaria ezin bada lortu edo denbora desberdintasuna 100ms baino handiagoa bada (monitorizazio sistemekiko neurturikoa). Puntu gehiago kentzen dira desberdintasun handiagoak direnean.

Grafikoa joerak ikusteko tresna gisa bakarrik dago pentsatua. Monitorizazio sistemak aurkitu duenari buruzko xehetasun zehatzagoetarako CSV estekara egin klik.

Offset grafikoa

Monitorizazio sistema gutxi gorabehera SNTP (RFC 2030) bezero bat bezala funtzionatzen du, beraz ntpd zerbitzari arrunt batek baino zerbitzariaren eta monitorizazio sistemaren arteko ausazko sare atzerapenei erantzun handiagoa du.

Monitorizazio sistema 10ms edo gehiago ere zehaztugabea izan daiteke.
