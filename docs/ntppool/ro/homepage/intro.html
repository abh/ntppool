Proiectul pool.ntp.org este un cluster virtual mare de servere de timp care oferă servicii NTP fiabile și ușor de folosit pentru milioane de clienți.

Pool-ul este folosit de sute de milioane de sisteme din întreaga lume. Este "serverul de timp" implicit pentru majoritatea distribuțiilor Linux majore și multe aparate în rețea (vezi informații pentru furnizori).

Din cauza numărului mare de utilizatori, avem nevoie de mai multe servere. Dacă aveți un server cu o adresă IP statică mereu disponibilă pe Internet, vă rugăm să considerați adăugarea lui în sistem.

Proiectul este întreținut și dezvoltat de Ask Bjørn Hansen și un grup minunat de colaboratori pe forumul comunității proiectului. Codul sursă pentru sistem este disponibil.

Hosting-ul și lățimea de bandă pentru serverele "hub" sunt oferite de Equinix și Netactuate.
