Graficul de scor

De câteva ori pe oră, sistemul pool-ului verifică timpul de pe serverul dumneavoastră și îl compară cu timpul său local. Puncte sunt deduse dacă serverul nu poate fi contactat sau dacă diferența de timp este mai mare de 100ms (măsurată din sistemele de monitorizare). Mai multe puncte sunt deduse cu cât diferența este mai mare.

Graficul este menit doar ca un instrument pentru vizualizarea tendințelor. Pentru detalii mai exacte despre ceea ce a găsit sistemul de monitorizare puteți da clic pe linkul CSV.

Graficul de diferență

Sistemul de monitorizare funcționează aproximativ ca un client SNTP (RFC 2030), astfel încât este mai susceptibil la latențele aleatorii de rețea între server și sistemul de monitorizare decât ar fi un server ntpd obișnuit.

Sistemul de monitorizare poate fi inexact cu până la 10ms sau mai mult.
