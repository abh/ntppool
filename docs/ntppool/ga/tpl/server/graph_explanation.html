An Scórghraf

Cúpla uair an chloig seiceálann an córas bailiúcháin an t-am ó do fhreastalaí agus cuireann sé é i gcomparáid leis an am áitiúil. Asbhaintear pointí murar féidir an freastalaí a bhaint amach nó má tá an t-am fritháirimh níos mó ná 100ms (arna thomhas ó na córais mhonatóireachta). Asbhaintear níos mó pointí is ea is mó an fhritháireamh.

Níl an graf i gceist ach mar uirlis chun treochtaí a shamhlú. Chun sonraí níos cruinne a fháil faoin méid a fuair an córas monatóireachta, is féidir leat cliceáil ar an nasc CSV.

An graf Fritháireamh

Oibríonn an córas monatóireachta a bheag nó a mhór cosúil le cliant SNTP (RFC 2030), agus mar sin bíonn sé níos so-ghabhálaí mar gheall ar fholaithe líonra randamach idir an freastalaí agus an córas monatóireachta ná mar a bheadh le gnáthfhreastalaí ntpd.

Is féidir leis an gcóras monatóireachta a bheith míchruinn chomh mór le 10ms nó níos mó.
