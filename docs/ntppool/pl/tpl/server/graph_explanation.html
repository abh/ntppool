Wykres punktacji

Kilka razy na godzinę system puli sprawdza czas z twojego serwera i porównuje go ze swoim lokalnym czasem. Punkty są odejmowane, jeśli serwer nie może być osiągnięty lub jeśli przesunięcie czasu jest większe niż 100ms (mierzone z systemów monitorowania). Więcej punktów jest odejmowanych, im większe jest przesunięcie.

Wykres ma służyć jedynie jako narzędzie do wizualizacji trendów. Aby uzyskać dokładniejsze szczegóły tego, co znalazł system monitorowania, możesz kliknąć na link CSV.

Wykres przesunięcia

System monitorowania działa mniej więcej jak klient SNTP (RFC 2030), więc jest bardziej podatny na losowe opóźnienia sieciowe między serwerem a systemem monitorowania niż regularny serwer ntpd.

System monitorowania może być niedokładny nawet o 10ms lub więcej.
