Monitor setup

The NTP Pool's monitoring infrastructure relies on dedicated volunteers from our community who run monitoring agents around the world. To ensure system reliability and maintain balanced coverage, our monitoring program operates by invitation only. If you're an active participant in the NTP Pool community, you can learn more about contributing through discussions at community.ntppool.org .

To install the monitoring agent, add the yum/dnf or apt/dpkg repository and install the ntppool-agent package.

Further instructions can be found in the (invitation-only) monitoring category on the community site.
