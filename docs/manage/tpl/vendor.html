Anyone distributing an appliance, operating system or some other kind of software using NTP needs a "vendor zone" if the default configuration is to use the pool system.

For internal use on systems you manage you do not need a vendor zone. For initial development or small-scale testing of an appliance or software you don't either, though remember to come back and get a vendor zone setup before production deployments or distribution start.

If you have any questions about this, please send questions before applying.
